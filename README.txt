🔥 NOVITÀ SISTEMA DI SALVATAGGIO:
✅ AUTO-SALVATAGGIO: I dati si salvano automaticamente ogni 30 secondi
✅ SALVATAGGIO ISTANTANEO: Ogni modifica viene salvata immediatamente
✅ CARICAMENTO AUTOMATICO: All'apertura dell'app, i dati vengono ricaricati automaticamente
✅ BACKUP SICURO: Pulsante "💾 Salva Dati" per creare file JSON di backup
✅ IMPORT/EXPORT: Pulsante "📁 Carica Dati" per importare file precedentemente salvati
✅ REPORT CSV: Pulsante "📊 Esporta CSV" per creare report Excel
✅ INDICATORE VISIVO: Notifica quando i dati vengono salvati automaticamente

🚀 COME USARE:
1. Estrai tutti i file in una cartella
2. Apri index.html con un browser web moderno (Chrome, Firefox, Safari, Edge)
3. L'applicazione funziona completamente offline
4. I dati vengono salvati automaticamente nel browser
5. Usa i pulsanti per backup e export

🔧 DATI TRACCIATI PER OGNI MESE:
- Chiamate partite, non risponde, non interessato, numero errato
- Appuntamenti fissati e percorsi
- Ore lavorate e costo orario
- Condominii potenziali (20 POD + 4 PDR)
- Potenziale PDR dichiarato agente (€850 ciascuno)
- Potenziale POD dichiarato agente (€160 ciascuno)

📈 METRICHE CALCOLATE AUTOMATICAMENTE:
- Chiamate effettive = Non interessato + Appuntamenti fissati
- ROI = (Potenziale acquisito - Costi) / Costi × 100
- Tassi di conversione per ogni fase del funnel
- Costo per appuntamento e valore medio
- KPI di performance completi

🎯 FUNNEL ANALYSIS:
Il sistema traccia l'intero percorso:
1. Chiamate partite
2. Chiamate effettive (non interessato + app preso)
3. Appuntamenti fissati
4. Appuntamenti percorsi
5. Totale acquisito potenziale (POD + PDR dichiarati dall'agente)
